# Business Requirements Document (BRD)
## Bella Rose Loyalty Program

### 1. Purpose
Design and implement a structured loyalty program to increase repeat revenue and customer lifetime value.

### 2. Scope
Includes:
- Tier-based membership
- Wallet credit system
- Staff incentive integration
- Dashboard reporting
- Campaign tracking

### 3. Functional Requirements

FR1: System must track annual customer spend  
FR2: System must auto-assign membership tier  
FR3: Wallet cashback must reflect in customer profile  
FR4: Dashboard must show repeat rate %  
FR5: Commission must be calculated based on membership conversion
